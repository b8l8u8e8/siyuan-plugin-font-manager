## ğŸš€ How to Use

1. Open **Font Manager** via the top bar icon or from the plugin settings.
2. Drag and drop font files into the upload area, or click "Browse Files" to **import fonts**.
3. **Select a font** from the list to apply it globally.
4. Use the bottom slider to **adjust font size**.
5. Click **Default System Font** to restore the original system settings.
