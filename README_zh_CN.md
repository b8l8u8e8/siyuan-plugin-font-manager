# ✨ 字体管理器

在思源内直接导入、管理和应用自定义字体。

📂 导入 → 🔤 选择 → ✅ 应用

🌍 文档语言：
[中文 README](https://github.com/b8l8u8e8/siyuan-plugin-font-manager/blob/main/README_zh_CN.md) ｜ [English README](https://github.com/b8l8u8e8/siyuan-plugin-font-manager/blob/main/README.md)

---

## 🎨 功能特性

| 功能         | 说明                                          |
| ------------ | --------------------------------------------- |
| 字体导入     | 拖放或浏览导入 TTF、OTF、WOFF、WOFF2 字体文件 |
| 全局应用     | 将导入的字体应用到整个思源界面                |
| 字体大小调节 | 滑块调节字体大小（12–24px）                   |

---

## 🚀 使用方法

1. 通过顶部栏图标或插件中打开**字体管理器**
2. 将字体文件拖放到上传区域，或点击「浏览文件」**导入字体**
3. 在字体列表中**选择字体**即可全局应用
4. 使用底部滑块**调节字体大小**
5. 点击**「默认（系统字体）」**恢复原始系统字体

---

## 📂 字体存储位置

导入的字体文件保存在：

- `/data/storage/petal/siyuan-plugin-font-manager/fonts`
